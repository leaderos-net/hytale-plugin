# LeaderOS Hytale Plugin

The official [LeaderOS](https://www.leaderos.net) plugin for Hytale servers. This plugin integrates your Hytale server with your LeaderOS website, enabling features like store purchases, account verification, credit management, and more.

## Building from Source

To build this project, you need **Java 21** and **Maven**.

**Prerequisites:**
You must have the `HytaleServer.jar` file located in the parent directory of this project (`/HytaleServer.jar`), as it is a system dependency.

1.  Clone the repository.
2.  Run the Maven build command:
    ```bash
    mvn clean install
    ```
3.  The compiled jar will be located in the `plugin/target` directory.
